# Java DevOps Project

## 📌 Overview
This is a simple Spring Boot application containerized using Docker.

## 🚀 Tech Stack
- Java 17
- Spring Boot
- Maven
- Docker
- GitHub

## ⚙️ Build & Run

### Build
mvn clean package

### Run locally
java -jar target/demo-0.0.1-SNAPSHOT.jar

### Docker Build
docker build -t java-app .

### Docker Run
docker run -p 8080:8080 java-app

## 🌐 Output
http://localhost:8080

## 📦 Features
- REST API endpoint
- Docker containerization
- GitHub integration
